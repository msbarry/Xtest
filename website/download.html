Download

Xtest 0.2.1 is out! See changelog for details.

Xtest requires Eclipse SDK 3.6 or higher and can be installed from one of the following update sites:

Official Releases: 
http://msbarry.github.com/Xtest/updates/release zip sha md5
Bleeding Edge: 
http://msbarry.github.com/Xtest/updates/develop zip sha md5

Within Eclipse, select Help -> Install New Software... the Add... and insert one of the update site URLs from above. Make sure that "Contact all update sites..." is selected. Click through the following screens and click OK when Eclipse warns you you are installing unsigned content, then restart Eclipse and Xtest will be installed!

See the documentation page for instructions on using Xtest. Don't hesitate to open a github issue for any problems found or suggestions.

Download update site as zip file

Alternatively, you can also download any of the above update sites as a zip file and from the Add... prompt in the Eclipse install window, click the Archive... button and point it to downloaded zip file.

If you already have Xtext installed

The above update sites will pull in the necessary Xtext dependencies if you do not have Xtext already installed. But if you do already have Xtext installed, choose the appropriate Xtest update site and version from the following compatibility matrix:
