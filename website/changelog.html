Changelog

0.2.1 (September 11, 2012)

Changes

  • Upgrade to Xtext 2.3.1 (tightened version dependencies to require exactly 2.3.1)
  • Convert build process to use Maven/Tycho
  • Update site now references Xtext so you don't need to add it manually
  • Restructured plugins into folders
  • New website
  • Miscellaneous bug fixes

0.2.0 (July 25, 2012)

Changes

  • Upgraded to Xtext 2.3.0
  • Numerous bug fixes
  • Added JUnit runner for Xtest files
  • Fixed performance bottlenecks found using YourKit

Language

  • Mark unexecuted expressions as warnings
  • Reusable methods declared using the def keyword
  • More detailed assertion failure messages
  • Filter invocation and internal Xtest elements out of stack traces

Eclipse IDE

  • Global and per-file runner configuration parameters
  • "Add Xtest libraries" shortcut
  • "Organize Imports" shortcut
  • Run tests on save in separate job after build completes
  • Run tests using recording class loader to track dependencies and only run tests affected by changes on save
  • Added Xtest status bar

0.1.1 (April 4, 2012)

Changes

  • Removed unneccessarily specific dependencies to fix installation issues
  • Back-ported Xtest for older installations of Xtext, available from separate update sites:

0.1.0 (March 20, 2012)

Initial release

Language

  • Support for test cases, test suites, and assertions
  • Type::class equivalent to Type.class in Java
  • := static assignment operator
  • Private members of Java classes are accessible
  • Import/import static support like Java

Eclipse IDE

  • Editor support and test-failure highlighting
  • Outline view shows test result structure
  • Navigator displays passing/failing state of Xtest files and their containers
  • New Xtest File wizard
